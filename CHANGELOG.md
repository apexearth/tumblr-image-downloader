# Changelog

## [1.0.0] - 2018-08-29

### Added
- `login` function to sign-in into Tumblr account for all requests.
- `getPhotos` function to retrieve photos for a given blog/page.
- `scrapeBlog` function retrieve all photos for a blog or range of pages in a blog.
